# Variant calling
## Index reference file
```
samtools faidx ${ref}.fasta
```

### Call variants

```
for sample in `cat sample_id.txt`
do
  freebayes -f ${ref.fasta} -p 1 Mapping/${sample}.clean.bam  > Mapping/${sample}.vcf
done
```

### Zip and index variants

```
bgzip ${vcf_file}
tabix ${vcf_file}.gz
```
